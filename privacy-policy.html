Overview

Who we are and what this extension does

YouTube Zoom Enabler is a Chrome extension developed and maintained by HVS. Its sole purpose is to provide keyboard-driven zoom and mouse-driven pan functionality for the YouTube video player on desktop browsers — bringing mobile-style zoom control to YouTube Watch pages and YouTube Shorts.

This Privacy Policy explains in clear, plain language what information the extension accesses, what it stores, what it never touches, and how we handle the minimal local data it does use.

💡 The short version: This extension zooms YouTube videos. It does not know who you are, what you watch, or anything else about you. That's it.
02

Data We Collect

The complete and exhaustive list

The extension stores exactly one piece of data on your device:

Data Value Type Where Stored Purpose Transmitted?
hvs_zoom Float (e.g. 2.25) chrome.storage.local
Your device only		 Remembers your last zoom level between sessions NEVER
🗂️ This number lives exclusively in your browser's local extension storage. It is not associated with any user account, device ID, or identity. It is never sent anywhere. It is simply a number between 1.0 and 9.0 that helps you avoid re-setting your zoom preference every time you open YouTube.
03

Data We Do Not Collect

Everything the extension deliberately never touches

The following data types are never collected, logged, transmitted, inferred, or processed in any form:

👤
Personal Identity No name, email, age, gender, or Google account information is ever accessed or stored.
📺
Watch History The extension never reads, logs, or transmits which videos you watch, for how long, or at what zoom level.
🌐
Browsing Data No URLs, page titles, search queries, or navigation history is accessed outside youtube.com.
📍
Location Data No IP address, GPS coordinates, timezone inference, or any geolocation data is collected.
⌨️
Keystrokes The extension only listens for U and Y keys, and only while not in a text field. It never logs keystrokes.
🖱️
Mouse Behaviour Mouse positions are used only for pan calculations in-memory. Nothing is recorded or stored.
📊
Usage Analytics No telemetry, crash reports, session counts, feature usage metrics, or any analytics are collected.
🍪
Cookies & Tokens The extension does not read, write, modify, or access any cookies, auth tokens, or session data.

Permissions Explained

Why each permission exists and what it cannot do

The extension requests the absolute minimum permissions required to function. Here is an honest, complete explanation of each:

activeTab
Allows the extension to inject the zoom engine and HUD overlay into the active YouTube tab. Without this, nothing works.
Scoped exclusively to youtube.com. Cannot access any other website, ever.
storage
Allows saving your zoom level preference (a single number) in your browser's local storage so it persists between sessions.
Only chrome.storage.local is used. Never chrome.storage.sync. Nothing is uploaded to any server.
🛡️ What is NOT requested: tabs, history, cookies, webRequest, identity, geolocation, downloads, notifications, or <all_urls>. None of these broader permissions are needed or used.

Third-Party Services

External resources the extension uses

The extension contains no third-party tracking scripts, advertising SDKs, analytics libraries, or data broker integrations of any kind. All functional code is self-contained within the installed extension package.

The only external resource loaded is a Google Fonts stylesheet used for the HUD overlay typography:

Service Purpose Data Sent User Data Involved?
Google Fonts Loads Space Mono and Syne typefaces for the HUD overlay display Standard HTTP font request (browser UA string only) NO
ℹ️ The Google Fonts request is a standard CSS font load — the same request any website makes when using Google Fonts. It carries no extension-specific data, no user identity, and no YouTube activity information. It can be blocked via browser settings without affecting core zoom functionality.
06

Local Storage

How and where data is stored on your device

The extension uses chrome.storage.local — a sandboxed key-value store that is part of Chrome's extension API. This storage is:

💾
Device-local The data exists only on the device where the extension is installed. It does not sync to other devices.
🔐
Sandboxed Only this extension can read or write to its own storage. Other extensions and websites cannot access it.
🗑️
Clearable by you You can delete all extension data at any time from Chrome's settings or by uninstalling the extension.
📦
Minimal Contains one key: hvs_zoom — a number. Total data footprint is under 20 bytes.

To manually clear this data: go to chrome://extensions → find YouTube Zoom Enabler → click Details → Storage → Clear data. Alternatively, uninstalling the extension removes all associated storage automatically.

Children's Privacy

Our stance on young users

YouTube Zoom Enabler does not collect any personal information from any user, of any age. Because the extension collects zero personal data by design, it is inherently safe for users of all ages, including children under 13.

The extension does not display advertisements, does not link to external content, does not create accounts, and does not require any form of registration or sign-in. There is no mechanism by which any user — child or adult — could submit personal information to us.
